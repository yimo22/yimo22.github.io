# Jakarta Contexts & Dependency Injection (CDI)
Jakarata EE의 규격을 맞춤, Spring 에 지원
CDI는 interface로서 구현체는 없고, Spring 이 이를 구현함
중요한 API 어노테이션으로는 다음과 같은 것들이 있다.
- Inject (~Autowired in spring)
- Named (~Component in Spring)
- Qualifier
- Scope
- Singleton


## 어노테이션

| Jakarta | Spring     |
| ------- | ---------- |
| @Named  | @Component |
| @Inject | @Autowired |
